## Setup
To run this project, follow the steps below:

```
$ cd portfolio
$ cp .env.example .env
$ composer install 
$ php artisan storage:link
$ php artisan passport:keys
$ php artisan key:generate
$ php artisan passport:install
$ npm install *
$ npm run watch
```
